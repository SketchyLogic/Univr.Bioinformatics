# Branch point

A single **adenosine residue** located ~18–40 bp upstream of the 3′ splice site (AG acceptor) within an intron. It is the catalytic nucleotide of the first step of splicing.

**Mechanism**: the 2′-OH group of the branch point adenosine performs a nucleophilic attack on the phosphodiester bond at the 5′ splice site. This breaks the 5′ exon–intron junction and forms a 2′–5′ phosphodiester bond between the branch point A and the first nucleotide of the intron, creating the **lariat intermediate** — a looped intron structure with the 5′ end tethered to an internal adenosine.

**Consensus sequence** (mammals): `YNYURAY`, where the last A is the reactive branch point adenosine. The sequence is loosely conserved — much less so than the GT-AG dinucleotides — which makes branch point detection computationally harder.

**Position in the 3′ splice site region**:

```
intron ... [branch point A] ── [polypyrimidine tract] ── AG | exon
                ~18–40 bp           ~15–40 bp           acceptor
```

See [[Polypyrimidine tract]] for the element immediately downstream.

# TLDR

The branch point is the adenosine inside the intron whose 2′-OH initiates splicing, forming the lariat. It sits ~18–40 bp upstream of the AG acceptor and has a loosely conserved consensus (YNYURAY).

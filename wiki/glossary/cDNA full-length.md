A cDNA sequence obtained by reverse-transcribing a **complete, mature mRNA** molecule, so that the resulting cDNA contains the full 5′ UTR, the entire CDS (from ATG start codon to stop codon), and the full 3′ UTR — as opposed to a partial fragment such as an [[Expressed Sequence Tag (EST)]].

**Why it is the gold standard for gene annotation**:
- Provides complete, experimentally verified gene structure
- Defines exact transcript boundaries (TSS at 5′ end, poly-A site at 3′ end)
- Contains a verified ORF with confirmed start and stop codons

**Contrast with EST**: An EST is a 400–800 bp partial fragment from one end of a cDNA; a full-length cDNA covers the entire transcript. Full-length cDNAs are fewer and more expensive to produce, so they are typically used as the reference/training set for annotation benchmarking.

In the *Vitis vinifera* benchmark, 3,436 non-redundant full-length cDNA sequences (with complete ORFs) were used: 936 as training data for *ab initio* predictors and 2,500 as the test/reference set for accuracy evaluation.
